# Ціна популізму

Інтерактивний калькулятор, який перетворює вартість державних програм в еквівалент військового забезпечення: дрони, снаряди, пікапи.

**[Відкрити калькулятор](https://velgaks.github.io/populism-price/)**

## Що це

Під час повномасштабної війни Україна витрачає десятки мільярдів гривень на програми, дизайн яких відповідає ознакам популістської фіскальної політики. Цей калькулятор показує альтернативну вартість цих витрат — скільки FPV-дронів, снарядів чи пікапів можна було б придбати за ті ж кошти.

Кожна програма оцінюється за [індексом популістськості](METHODOLOGY.md) — 7 бінарних критеріїв, заснованих на академічній літературі з фіскальної політики.

## Методологія

Детальний опис індексу популістськості, критеріїв та академічних джерел — у файлі **[METHODOLOGY.md](METHODOLOGY.md)**.

## Технології

- React (один `.jsx` файл) + Tailwind CSS
- Vite для збірки
- GitHub Pages для хостингу

## Розробка

```bash
npm install
npm run dev       # localhost:5173
npm run build     # збірка в dist/
```

## Як додати нову програму

1. Додати об'єкт у масив `DATA.programs` в `src/App.jsx`
2. Оцінити за 7 критеріями індексу (див. [METHODOLOGY.md](METHODOLOGY.md))
3. Оновити `DATA.context.totalProgramsCostUAH`
4. Оновити `DATA.meta.lastUpdated`

## Дисклеймер

Дані є наближеними оцінками для ілюстративних цілей. Ми не стверджуємо, що кожна програма — «погана». Деякі мають реальний соціальний ефект. Калькулятор оцінює, наскільки дизайн програми відповідає ознакам популістської політики, і показує альтернативну вартість у контексті війни.
