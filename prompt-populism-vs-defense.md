# Промпт: «Ціна популізму» — інтерактивний калькулятор

## Контекст для AI

Ти — фронтенд-розробник і дата-журналіст. Потрібно створити **одно-файловий React-додаток (.jsx)** українською мовою — інтерактивний калькулятор, який перетворює вартість популістських/неефективних державних програм в Україні на еквівалент у військовому забезпеченні (дрони, снаряди, тощо).

**Цільова аудиторія:** широка публіка + соцмережі. Має бути максимально простим, візуальним, емоційним — і при цьому спиратися на верифіковані дані з посиланнями.

## UI/UX вимоги

### Загальна концепція
Сторінка — **емоційний калькулятор**: вибираєш програму → бачиш скільки дронів, снарядів, пікапів можна було б купити.

### Структура сторінки (один екран, скрол)

**1. Hero-блок**
- Назва проєкту: **«Ціна популізму»** — велика, характерна типографіка
- Підзаголовок: «Скільки дронів, снарядів і пікапів можна купити замість державних роздач»
- Контекст: «Дефіцит військових видатків — ~400 млрд грн» (Железняк, 12.03.2026)
- **Живий лічильник** (головна фіча hero-блоку):
  - Формула: `annualCostUAH / (365 * 24 * 60 * 60)` = грн/сек для обраної програми
  - Відображення: великий анімований лічильник «₴ XX XXX витрачено, поки ви на цій сторінці»
  - Стартує з 0, тікає в реальному часі (requestAnimationFrame або setInterval 100ms)
  - Під лічильником — дрібний текст «це ще N FPV-дронів» (оновлюється разом з лічильником)
  - При зміні програми — скидається на 0
  - Технічна реалізація: `useEffect` з `setInterval`, зберігати startTime в `useRef`

**2. Блок вибору програми**
- Великі кнопки/картки з назвами програм
- Активна підсвічується
- Сума (напр. «₴14.4 млрд») + посилання на джерело
- Кнопка «УСЬОГО: ₴86.3 млрд» для сумарного еквіваленту

**3. Блок результатів — «На цю суму можна купити:»**
- Сітка карток з озброєнням
- Emoji + назва + **велике число** кількості одиниць
- Візуалізація масштабу (іконки, до ~100 шт)
- Коротке пояснення (напр. «це N FPV-дронів на кожен день року»)
- Count-up анімація при перемиканні

**4. Порівняльний контекст**
- «Це покриває Y% дефіциту військових видатків»
- «За один день цієї програми — X дронів»

**5. Методологія і джерела (footer)**
- Згорнутий блок, посилання, дисклеймер
- **Авторство:** «Валентин Гацко · KSE Center for Sociological Research»
- **Посилання на GitHub:** іконка + лінк на репозиторій (заповнити пізніше, поки placeholder `https://github.com/...`)
- Текст під лінком: «Дані та код — відкриті. Знайшли помилку? Створіть issue.»

**6. Шер-картка**
- Кнопка «Поділитися» → копіює текст: «Ціна популізму: [програма] = ₴X = [N] FPV-дронів. Перевір: [url]»

### Дизайн
- **Тон:** серйозний, документальний, як Forbes/VoxUkraine
- **Палітра:** темний фон (графіт), жовтий/золотий акцент, білий текст
- **Типографіка:** Unbounded (заголовки) + Montserrat (текст), Google Fonts
- **Mobile-first**, основна аудиторія — телефон через соцмережі
- **Анімації:** count-up, плавна поява карток

## Технічні вимоги

- **Один .jsx файл** (React)
- `const DATA = {...}` — на початку файлу
- Tailwind utility classes
- Ніякого localStorage — тільки useState
- Числа — по-українськи (пробіл як роздільник тисяч)
- Всі тексти — українською

### Як додати нову програму

Додати об'єкт у `DATA.programs`:
```js
{
  "id": "unique-id",
  "name": "Назва",
  "description": "Опис",
  "annualCostUAH": 5000000000,  // грн, без форматування
  "year": "2026",
  "sourceLabel": "Джерело",
  "sourceUrl": "https://...",
  "notes": ""
}
```
Оновити `context.totalProgramsCostUAH` і `meta.lastUpdated`.
